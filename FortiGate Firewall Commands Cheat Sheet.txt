## Basic FortiGate Firewall CLI Commands

### System Information
get system status
get hardware status
get system performance status


### Check Interface IPs
show system interface
get system interface


### Ping Test
execute ping 8.8.8.8
execute ping-options source <interface-ip>


### Traceroute
execute traceroute google.com


### DNS Check


get system dns
execute nslookup google.com




# Configuration Commands

## Configure Hostname


config system global
set hostname FortiGate
end


## Configure Interface IP


config system interface
edit port1
set ip 192.168.1.1 255.255.255.0
set allowaccess ping https ssh
next
end


## Configure Default Gateway


config router static
edit 1
set gateway 192.168.1.254
set device port1
next
end


## Configure DNS


config system dns
set primary 8.8.8.8
set secondary 1.1.1.1
end




# Firewall Policy Commands

## View Firewall Policies


show firewall policy


## Create Firewall Rule


config firewall policy
edit 1
set name "LAN_to_WAN"
set srcintf "port2"
set dstintf "port1"
set srcaddr "all"
set dstaddr "all"
set action accept
set schedule "always"
set service "ALL"
set nat enable
next
end




# User & Admin Commands

## Create Admin User


config system admin
edit admin2
set password password123
next
end


## Show Admin Users


show system admin




# Routing Commands

## Show Routing Table


get router info routing-table all


## Add Static Route


config router static
edit 1
set dst 0.0.0.0/0
set gateway 192.168.1.1
set device port1
next
end




# VPN Commands

## Show VPN Status


get vpn ipsec tunnel summary
diagnose vpn tunnel list


## Restart VPN Tunnel


diagnose vpn tunnel flush




# NAT Commands

## Enable NAT in Policy


set nat enable


## View Central NAT


show firewall central-snat-map




# DHCP Commands

## Configure DHCP Server


config system dhcp server
edit 1
set interface port2
set lease-time 86400
config ip-range
edit 1
set start-ip 192.168.10.10
set end-ip 192.168.10.100
next
end
next
end


## Check DHCP Leases


execute dhcp lease-list
